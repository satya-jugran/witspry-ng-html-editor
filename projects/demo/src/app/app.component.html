Features

  • ✅ Auto-indentation
  • ✅ Multiple Themes (Default, Light, Dark)
  • ✅ Configurable Tab Size
  • ✅ Configurable Font Size
  • ✅ Word Wrap Support
  • ✅ Read-only Mode
  • ✅ NgModel Integration
  • ✅ Event Handling (focus, blur, content change)

Usage Example

<witspry-html-editor
  [(ngModel)]="htmlContent"
  [theme]="'dark'"
  [config]="editorConfig"
  [readonly]="false"
  placeholder="Enter HTML content..."
  (contentChange)="onContentChange($event)"
  (focus)="onEditorFocus()"
  (blur)="onEditorBlur()">
</witspry-html-editor>
